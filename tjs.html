JavaScript Concepts

1. Basic Concepts

            
  • var, let, const: Keywords to declare variables.
    • 
        
  • console.log(): Outputs messages to the browser console.
    • 
        
  • alert(): Displays a browser alert dialog.
    • 
        
  • typeof: Returns the data type of a variable or value.
    • 
        
  • if, else, else if: Conditional statements to execute code based on conditions.
    • 
        
  • for, while, do...while: Loops to iterate over blocks of code.
    •

2. Functions

            
  • function: Declares a reusable block of code.
    • 
        
  • arrow functions: A shorter syntax for writing functions (() => {}).
    • 
        
  • return: Specifies the value a function will return.
    • 
        
  • parameters and arguments: Input values passed to a function.
    •

3. DOM Manipulation

            
  • document.getElementById(): Selects an element by its ID.
    • 
        
  • document.querySelector(): Selects the first matching element using a CSS selector.
    • 
        
  • document.createElement(): Creates a new HTML element.
    • 
        
  • element.innerHTML: Sets or gets the HTML content of an element.
    • 
        
  • element.style: Applies CSS styles to an element.
    • 
        
  • element.addEventListener(): Attaches an event handler to an element.
    •

4. Events

            
  • click: Triggered when an element is clicked.
    • 
        
  • keyup and keydown: Triggered when a key is pressed or released.
    • 
        
  • mouseover: Triggered when the mouse pointer enters an element.
    • 
        
  • submit: Triggered when a form is submitted.
    •

5. Objects

            
  • Object: A collection of key-value pairs.
    • 
        
  • this: Refers to the current object context.
    • 
        
  • Object.keys(): Returns an array of an object's property names.
    • 
        
  • Object.values(): Returns an array of an object's values.
    •

6. Arrays

            
  • push(): Adds an item to the end of an array.
    • 
        
  • pop(): Removes the last item of an array.
    • 
        
  • shift(): Removes the first item of an array.
    • 
        
  • unshift(): Adds an item to the beginning of an array.
    • 
        
  • forEach(): Calls a function for each array item.
    • 
        
  • map(): Creates a new array by applying a function to each item.
    • 
        
  • filter(): Creates a new array with items that pass a condition.
    •

7. ES6 Features

            
  • let and const: Block-scoped variable declarations.
    • 
        
  • template literals: String interpolation using backticks (`Hello, ${name}`).
    • 
        
  • spread operator: Expands an array or object (...).
    • 
        
  • destructuring: Extracts values from arrays or objects (const {x, y} = obj;).
    • 
        
  • modules: Import and export functionality between files (import/export).
    •

8. Promises and Asynchronous Programming

            
  • Promise: Represents a value that may be available now, later, or never.
    • 
        
  • async and await: Simplify asynchronous code.
    • 
        
  • then(): Handles a resolved promise.
    • 
        
  • catch(): Handles errors in promises.
    •

9. Errors and Debugging

            
  • try...catch: Handles errors in a block of code.
    • 
        
  • throw: Throws a custom error.
    • 
        
  • console.error(): Outputs error messages to the console.
    •

10. Browser APIs

            
  • fetch(): Sends HTTP requests and handles responses.
    • 
        
  • localStorage and sessionStorage: Store data in the browser.
    • 
        
  • setTimeout(): Executes code after a delay.
    • 
        
  • setInterval(): Executes code repeatedly at specified intervals.
    • 
        
  • navigator.geolocation: Retrieves the user's location.
    •

11. Classes

            
  • class: Defines a blueprint for creating objects.
    • 
        
  • constructor(): Initializes class properties.
    • 
        
  • methods: Functions inside a class.
    • 
        
  • extends: Inherits properties and methods from another class.
    • 
        
  • super(): Calls the constructor of the parent class.
    •
