What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design pages on the web. It structures content into headings, paragraphs, links, images, lists, and other elements to display on web browsers.

How HTML Works

HTML consists of a series of elements (tags) that define different parts of a webpage. Tags are enclosed in angle brackets like <tag> and most HTML elements come in pairs: an opening tag and a closing tag. The closing tag has a slash before the tag name, such as </tag>.

Basic HTML Structure

Every HTML document has a basic structure that includes the <html> tag as the root, the <head> section for metadata, and the <body> section for content.

Example:

       <!DOCTYPE html>
       <html lang="en">
       <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <title>My First Web Page</title>
       </head>
       <body>
      <h1>Hello, World!</h1>
      <p>This is my first HTML webpage.</p>
      </body>
      </html>

HTML Basic Command

HTML Command Description Example
<html> Defines the root of an HTML document. All content is placed inside the <html> tag.
<html> ... </html>
<head> Contains metadata about the HTML document, such as the title and link to stylesheets.
<head> </head>
<body> Defines the body of the document, where all visible content such as text, images, and other elements go.
<body> Content here... </body>
<h1> Defines the largest heading. Headings range from <h1> to <h6>, with <h1> being the most important.
<h1>This is a Heading 1</h1>
<p> Defines a paragraph of text.
<p>This is a paragraph.</p>
<a> Defines a hyperlink. It can be used to link to another webpage or location within the same page.
<a href="https://www.example.com">Visit Example</a>
<img> Defines an image element. Requires the src attribute to specify the image file.
<img src="image.jpg" alt="Description">
<ul> Defines an unordered list. The list items inside the <ul> are typically marked with bullets.
<ul> <li>Item 1</li> <li>Item 2</li> </ul>
<ol> Defines an ordered list. The list items inside the <ol> are typically numbered.
<ol> <li>First item</li> <li>Second item</li> </ol>
<div> Defines a division or section in a document. It's used as a container for other elements.
<div> Content here... </div>
<form> Defines an HTML form for user input. It can contain various types of input fields, buttons, and labels.
<form action="submit.php"> ... </form>
<input> Defines an input field in a form. The type attribute determines the type of input (text, password, checkbox, etc.).
<input type="text" placeholder="Enter text">
<label> Defines a label for an input element. It is typically used with id attributes to provide descriptions for form controls.
<label for="name">Name</label>
<table> Defines a table. It contains rows and columns to organize content.
<table> ... </table>
<tr> Defines a table row.
<tr> ... </tr>
<td> Defines a table cell, containing data.
<td>Data</td>
<th> Defines a table header cell. It is typically used to define headings in a table.
<th>Header</th>
<audio> Defines an audio file to be played. It can include controls like play, pause, and volume.
<audio controls><source src="audio.mp3"></audio>
<video> Defines a video file to be played. It can include controls like play, pause, and volume.
<video controls><source src="video.mp4"></video>
<link> Defines the relationship between the HTML document and an external resource (e.g., a stylesheet).
<link rel="stylesheet" href="styles.css">
<meta> Defines metadata about the document, such as the character set, author, or keywords.
<meta charset="UTF-8">
