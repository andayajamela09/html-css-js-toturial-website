HTML Commands

1. Document Structure

    
        
  • <!DOCTYPE html>: Defines the type of document (HTML5).
    • 
        
  • <html>: Root element of the HTML document.
    • 
        
  • <head>: Contains metadata (e.g., title, styles, scripts).
    • 
        
  • <title>: Sets the title displayed in the browser tab.
    • 
        
  • <body>: Contains the visible content of the web page.
    •

2. Text Formatting and Structure

    
    
        
  • <h1> to <h6>: Headings, from largest (<h1>) to smallest (<h6>).
    • 
        
  • <p>: Defines a paragraph.
    • 
        
  • <br>: Inserts a line break.
    • 
        
  • <hr>: Adds a horizontal line for separation.
    • 
        
  • <strong>: Highlights text with importance (bold).
    • 
        
  • <em>: Adds emphasis to text (italic).
    • 
        
  • <b>: Makes text bold (no semantic meaning).
    • 
        
  • <i>: Makes text italic (no semantic meaning).
    • 
        
  • <mark>: Highlights text.
    • 
        
  • <small>: Makes text smaller.
    • 
        
  • <sup>: Displays superscript text.
    • 
        
  • <sub>: Displays subscript text.
    • 
        
  • <del>: Shows deleted text (strikethrough).
    • 
        
  • <ins>: Indicates inserted text (underlined).
    •

3. Links and Navigation

    
    
        
  • <a href="URL">: Creates a hyperlink.
    • 
        
  • <nav>: Groups navigation links.
    •

4. Lists

    
    
        
  • <ul>: Creates an unordered list.
    • 
        
  • <ol>: Creates an ordered list.
    • 
        
  • <li>: List item (used inside <ul> or <ol>).
    • 
        
  • <dl>: Creates a definition list.
    • 
        
  • <dt>: Term in a definition list.
    • 
        
  • <dd>: Description or definition for a term.
    •

5. Multimedia

    
    
        
  • <img src="URL" alt="description">: Embeds an image.
    • 
        
  • <audio>: Embeds audio content.
    • 
        
  • <video>: Embeds video content.
    • 
        
  • <embed>: Embeds external resources (e.g., videos, maps).
    • 
        
  • <object>: Embeds objects like multimedia or applications.
    •

6. Tables

    
    
        
  • <table>: Creates a table.
    • 
        
  • <tr>: Table row.
    • 
        
  • <td>: Table data cell.
    • 
        
  • <th>: Table header cell.
    • 
        
  • <thead>: Groups the header rows in a table.
    • 
        
  • <tbody>: Groups the body rows in a table.
    • 
        
  • <tfoot>: Groups the footer rows in a table.
    • 
        
  • <caption>: Adds a title or description to the table.
    • 
        
  • <colgroup>: Defines groups of columns in a table.
    • 
        
  • <col>: Styles individual columns.
    •

7. Forms

    
    
        
  • <form>: Defines a form for user input.
    • 
        
  • <input>: Single-line text field or other input controls.
    • 
        
  • <textarea>: Multi-line text input.
    • 
        
  • <button>: Clickable button.
    • 
        
  • <label>: Labels a form element for better accessibility.
    • 
        
  • <select>: Drop-down menu.
    • 
        
  • <option>: Items in a drop-down menu.
    • 
        
  • <fieldset>: Groups form elements visually.
    • 
        
  • <legend>: Caption for a <fieldset>.
    •

8. Semantic Elements

    
    
        
  • <header>: Defines a header for a document or section.
    • 
        
  • <footer>: Defines a footer for a document or section.
    • 
        
  • <main>: Specifies the main content of the document.
    • 
        
  • <section>: Groups related content.
    • 
        
  • <article>: Represents a self-contained piece of content.
    • 
        
  • <aside>: Sidebar or related content.
    • 
        
  • <div>: Generic container for content.
    • 
        
  • <span>: Inline container for content.
    •

9. Script and Metadata

    
    
        
  • <script>: Embeds or links JavaScript code.
    • 
        
  • <style>: Embeds CSS styles.
    • 
        
  • <link>: Links external stylesheets or resources.
    • 
        
  • <meta>: Provides metadata about the HTML document.
    •

10. Interactive Elements

    
    
        
  • <details>: Creates a collapsible section.
    • 
        
  • <summary>: Title for a collapsible section.
    • 
        
  • <dialog>: Creates a dialog box or modal.
    • 
        
  • <canvas>: For drawing graphics using JavaScript.
    • 
        
  • <svg>: For scalable vector graphics.
    •
